# Okane

Okane is a financial management application built using C# and TypeScript. It provides users with tools to track their expenses and analyze their financial data.

## Main Goal

Okane aims to be very simple and straightforward by removing all the unnecessary features that other financial management applications have. 

The main goal is to provide users with a clean and intuitive interface that allows them to easily manage their finances without being overwhelmed by too many options.
